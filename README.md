# Equipment Reliability & Maintenance Analytics Dashboard

## Project Overview

This project analyzes industrial machine performance, component failures, maintenance activities, and sensor readings using SQL Server and Power BI. The goal of the analysis is to monitor equipment reliability, identify operational trends, and support maintenance decision-making through interactive dashboards and KPI reporting.

The dataset used simulates real-world industrial maintenance operations and includes telemetry sensor data, machine failures, maintenance records, and error logs.

# Key Insights

* Component 2 recorded the highest failure frequency across the dataset.
* Model 3 machines experienced the highest number of failures.
* Failure trends fluctuated monthly, indicating operational variability.
* Higher vibration readings were observed in several frequently failing machines.
* Older machines did not always produce the highest failure counts, suggesting operational conditions also influence reliability.
